# 批量邮件发送工具 - 企业微信邮箱版

通过企业微信邮箱 SMTP 批量发送个性化邮件，支持 Excel 名单导入和模板变量替换。

## 快速启动

```bash
cd email-sender
pip install -r requirements.txt
python app.py
```

浏览器打开 http://localhost:5050 即可使用。

## 使用步骤

1. **配置 SMTP** - 填写企业微信邮箱地址和密码，点击「测试连接」确认
2. **上传名单** - 上传包含邮箱列的 Excel 文件（.xls / .xlsx），选择邮箱所在列
3. **编辑邮件** - 选择模板或自定义 HTML 正文，用 `{{列名}}` 插入个性化变量
4. **批量发送** - 确认无误后点击发送，实时查看进度和结果

## 企业微信邮箱 SMTP 配置

| 配置项 | 值 |
|--------|-----|
| SMTP 服务器 | smtp.exmail.qq.com |
| 端口 | 465 (SSL) |
| 密码 | 邮箱登录密码 或 客户端专用密码 |

> 客户端专用密码在企业微信邮箱「设置 → 邮箱绑定 → 安全登录」中生成。

## 模板变量

邮件主题和正文中可使用 `{{列名}}` 格式的变量，发送时会自动替换为 Excel 中对应行的值。

例如 Excel 中有「姓名」「邮箱」列，正文中写 `Hi {{姓名}}` 会自动替换为每个收件人的姓名。
